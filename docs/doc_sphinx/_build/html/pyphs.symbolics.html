class pyphs.symbolics.expressions.Expressions(phs)¶

Class that serves as a container for all symbolic expressions * H: Hamiltonian function expressions * z: tuple of dissipation functions expressions * g: tuple of input/output gains functions expressions

build()¶
Build the following system functions as sympy expressions and append them as attributes to the exprs module:
  • ‘dxH’ the continuous gradient vector of storage scalar function exprs.H,
  • ‘dxHd’ the discrete gradient vector of storage scalar function exprs.H,
  • ‘hessH’ the continuous hessian matrix of storage scalar function exprs.H,
  • ‘jacz’ the continuous jacobian matrix of dissipative vector function exprs.z,
  • ‘y’ the continuous output vector function,
  • ‘yd’ the discrete output vector function.
freesymbols()¶

Retrun a set of freesymbols in all exprs in _names

setexpr(name, expr)¶

Add the sympy expression ‘expr’ to the exprs module, with argument ‘name’, and add ‘name’ to the set of _names

class pyphs.symbolics.symbols.Symbols¶

Class that serves as a container for all symbols: * x: list of state variable symbols * w: list of dissipation variable symbols * u: list of input symbols * y: list of output symbols * p: list of parameter symbols * subs: dictionary with symbols as keys and substitution exprs as values * cu: list of connectors input symbols * cy: list of connectors outputs symbols * _names: set of names for all the lists

args()¶

return list of symbols associated with arguments of numerical functions

dx()¶

returns the states increment symbols “dxi” associated with state symbol “xi” for each “xi” in state vector “x”.

pyphs.numerics.discrete_calculus.discrete_gradient(H, x, dx, numtol=2.2204460492503131e-16)¶

Symbolic computation here. Return the discrete gradient of scalar function H between x and x+dx, with H separable with respect to x: H = sum_i[H_i(x_i)].

H : sympy.expression

Scalar function of x.

x : list or sympy.Matrix

1 dimensional array of sympy symbols.

dx : list or sympy.Matrix

1 dimensional array of sympy symbols.

gradd : list

discrete gradient of H with ‘gradd[i] = H.diff(x) if norm(dx[i]) < numtol else (H(x+dx[i])-H(x))/dx[i]’
